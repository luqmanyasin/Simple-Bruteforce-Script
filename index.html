Simple-Bruteforce-Script

A Simple Bruteforce Script that runs using PHP and JavaScript

This Script will, once completed run a set of characters through a selected form, through either an action or AJAX, depending on the script. It will then detect any submission it deems to be possibly successful.

Live Preview: http://drogon.rawhost.net/~thinkte/rand-as/bruteforce/

For Live Preview Use The Following Example Inputs:

  • Method: Post
  • Input Name: code
  • Action URL: post.php
  • Values: a,b,c,d
  • Username: exampleUser
  • User Form Name: user
